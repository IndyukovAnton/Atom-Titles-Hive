# Web Titles Tracker

Платформа для отслеживания просмотров видео, фильмов, аниме и другого контента с возможностью группировки и аналитики.

## Особенности

- Локальное приложение (запускается как desktop-приложение)
- Адаптивный дизайн
- Группировка записей
- Статистика просмотров
- Поиск и фильтрация
- Управление профилем пользователя

## Технологии

- Next.js 19+
- React
- React Router
- SQLite
- Electron
- Tailwind CSS

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите в режиме разработки:
```bash
npm run dev
```

3. Соберите для production:
```bash
npm run build
```

4. Запустите desktop-приложение:
```bash
npm run electron
```

## Структура проекта

- `src/app` - Основные страницы приложения
- `src/components` - Повторно используемые компоненты
- `server` - Бэкенд на Express.js
- `data` - База данных SQLite
- `logs` - Лог-файлы
- `uploads` - Загруженные изображения
- `electron` - Конфигурация Electron

## Функциональность

### Главная страница
- Отображение всех записей
- Поиск и фильтрация
- Группировка по категориям
- Форма создания новой записи

### Страница профиля
- Информация о пользователе
- Статистика просмотров
- Любимые категории и жанры

### Настройки
- Настройки аккаунта и сайта

### Авторизация
- Регистрация и вход
