# Проект

Проект представляет собой веб-приложение для отслеживания тайтлов (фильмов, сериалов, аниме, книг и т.д.)

В проекте должен быть реализован функционал:
- Регистрация и авторизация
- Создание групп
- Добавление тайтлов в группы
- Управление тайтлами
- Управление группами
- Управление пользователями
- Управление настройками
- Управление аутентификацией
- Управление авторизацией

Мобильное приложение должно быть реализовано на React Native.
Десктопное приложение должно быть реализовано на Tauri.
Веб-приложение должно быть реализовано на React.
Серверная часть должна быть реализована на NestJS.

Весь UI в проекте должен быть реализован с использованием shadcn-компонентов.
