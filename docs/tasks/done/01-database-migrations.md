# Задача BE-01: Настройка миграций базы данных

## Зависимости
- Нет зависимостей

## Описание проблемы
В `backend/src/app.module.ts` используется `synchronize: true`, что допустимо только для разработки. Для production необходимо использовать миграции TypeORM.

Текущие риски:
- Потеря данных при изменении схемы
- Невозможность отката изменений
- Отсутствие контроля версий схемы БД
- Нарушение best practices TypeORM

## Цель задачи
Настроить систему миграций TypeORM для управления схемой базы данных в production.
