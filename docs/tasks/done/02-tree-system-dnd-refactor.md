# Задача FE-02: Улучшение системы Дерева Групп (DnD и UX)

## Зависимости
- FE-02: Текущая реализация компонента `Tree` в `src/components/ui/tree.tsx`

## Описание проблемы
Текущая система управления группами неудобна:
1. Отсутствует Drag & Drop (перетаскивание) групп.
2. Невозможно перетаскивать записи из одной группы в другую.
3. При добавлении подгруппы дерево сворачивается, что сбивает пользователя.
4. Удаление в контекстном меню не работает.

## Цель задачи
Реализовать полноценную систему Drag & Drop для групп и записей, исправить ошибки UX в дереве.

## Чек-лист выполнения

### Этап 1: Drag & Drop (DnD)
- [x] Интегрировать поддержку перетаскивания в компонент `Tree` (используя `@headless-tree/core` или `dnd-kit`).
- [x] Реализовать перетаскивание групп для изменения вложенности или порядка.
- [x] Реализовать перетаскивание записей (Media Items) в группы внутри дерева.
- [x] Добавить визуальную индикацию места сброса (drop indicator).

### Этап 2: UX и Состояние дерева
- [x] Исправить поведение сворачивания: при добавлении подгруппы родительская группа должна оставаться развернутой.
- [x] Обеспечить сохранение состояния (expanded/collapsed) в LocalStorage или глобальном стейте.

### Этап 3: Багфиксы и API
- [x] Починить функцию удаления в контекстном меню (проверить передачу ID и вызов хука удаления).
- [x] Исправить Warning: "Select is changing from uncontrolled to controlled" в фильтрах/выборе групп.
- [x] Добавить подтверждение удаления для групп.

## Критерии приёмки
- ✅ Группы перетаскиваются и сохраняют новую позицию в БД.
- ✅ Записи можно перетаскивать в группы.
- ✅ Дерево не сворачивается самопроизвольно при операциях.
- ✅ Контекстное меню работает корректно.
- ✅ Отсутствуют React-варнинги в консоли при работе с деревом.
