# Задача FE-03: Редизайн формы добавления контента (Add Media Modal)

## Зависимости
- FE-03: Текущий компонент `AddMediaModal.tsx`

## Описание проблемы
Текущая форма добавления записи перегружена инпутами, выглядит неэстетично и неудобна в использовании. Не хватает интерактивности, визуального предпросмотра и удобного выбора тегов/критериев.

## Цель задачи
Полностью переработать форму добавления в стиле "Premium", добавив пошаговость, табы и современные виджеты.

## Чек-лист выполнения

### Этап 1: Структура и UX
- [x] Разделить форму на тематические этапы (Этап 1: Основная инфо, Этап 2: Детали и критерии, Этап 3: Медиа и изображения).
- [x] Внедрить систему табов или горизонтальный слайдер (свайп) для переключения этапов.
- [x] Использовать React Portals для рендеринга модального окна (исправление лагов и наложений).
- [x] Оптимизировать рендеры: убрать пролагивания при открытии и вводе данных.

### Этап 2: Интерактивные элементы
- [x] **Рейтинг**: Создать кастомный виджет выбора рейтинга (звезды/числа) с плавной анимацией.
- [x] **Дата**: Внедрить красивый DatePicker (календарь), подходящий под дизайн.
- [x] **Теги и Критерии**: Реализовать систему в стиле Notion/Obsidain:
  - [x] Список существующих тегов с поиском.
  - [x] Возможность создать новый тег прямо из поля ввода.
  - [x] Удобное удаление и редактирование добавленных тегов.
- [x] **Категории**: Добавить возможность создать новую категорию (группу), если нужная отсутствует в списке.

### Этап 3: Загрузка медиа
- [x] Добавить визуальный предпросмотр (Preview) выбранных изображений.
- [x] Реализовать возможность удаления выбранного изображения до отправки формы.
- [x] Добавить индикатор прогресса загрузки.

## Критерии приёмки
- ✅ Форма выглядит современно и "дорого", нет нагромождения полей.
- ✅ Переключение между этапами происходит мгновенно и плавно.
- ✅ Теги добавляются и удаляются удобно (без лишних кликов).
- ✅ Пользователь видит, какие картинки он выбрал.
- ✅ Модалка открывается через Portal.
