# Задача BE-03: Реализация модуля статистики

## Зависимости
- Нет зависимостей

## Описание проблемы
Согласно README.md и requirements, ProfilePage должен отображать подробную статистику:
- Общее количество записей
- Любимая категория (наиболее частая)
- Любимый жанр
- Общее время просмотра
- Время просмотра за текущий месяц

На данный момент `profile.service.ts` возвращает только базовую информацию о пользователе без статистики.

## Цель задачи
Расширить ProfileService для вычисления и возврата подробной статистики пользователя.
