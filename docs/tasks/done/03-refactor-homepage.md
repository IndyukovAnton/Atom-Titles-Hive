# Задача 03: Рефакторинг HomePage.tsx

## Описание проблемы
Компонент `HomePage.tsx` содержит 275 строк кода, где смешаны:
- Бизнес-логика (загрузка данных, обработка drag-and-drop)
- UI логика (рендеринг групп, медиа-карточек)
- Управление состоянием (media, groups, модалки, контекстное меню)
- Обработчики событий

Это нарушает принципы:
- Single Responsibility Principle
- Separation of Concerns
- Тестируемость кода

## Цель задачи
Разбить `HomePage.tsx` на логические модули: кастомные хуки для логики, подкомпоненты для UI.
