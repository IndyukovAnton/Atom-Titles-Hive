# Задача 04: Вынос API URL в переменные окружения

## Описание проблемы
В файле `src/api/client.ts` захардкожен URL API:
```typescript
const API_URL = 'http://localhost:1221';
```

Это создаёт проблемы:
- Невозможно переключаться между dev/staging/production окружениями
- Нельзя запустить на другом порту без изменения кода
- Сложно настраивать CI/CD
- Нарушает best practices (конфигурация должна быть вне кода)

## Цель задачи
Вынести все конфигурационные параметры в переменные окружения и создать систему управления конфигурацией.

## Чек-лист выполнения

- [x] Создать `.env.development` для dev окружения
- [x] Создать `.env.production` для production окружения
- [x] Создать `.env.example` с примерами всех переменных
