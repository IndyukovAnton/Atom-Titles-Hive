# Задача BE-04: Написание Unit и Integration тестов

## Зависимости
- Нет зависимостей (можно выполнять параллельно с другими)

## Описание проблемы
В backend отсутствуют тесты (кроме заглушки `app.controller.spec.ts`). Для надёжности и поддерживаемости кода необходимо покрытие тестами:
- Unit тесты для services
- Integration тесты для controllers
- E2E тесты для критичных user flows

## Цель задачи
Создать полное покрытие тестами для всех модулей backend (auth, media, groups, profile).
