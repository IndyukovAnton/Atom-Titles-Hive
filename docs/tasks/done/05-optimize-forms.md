# Задача 05: Оптимизация работы с формами ✅

## Описание проблемы
В компоненте `AddMediaModal.tsx` (239 строк):
- Каждое изменение input вызывает отдельный `setFormData`, что приводит к множественным ре-рендерам
- Отсутствует централизованная валидация формы
- Дублируется логика обработки изменений для разных полей
- Нет переиспользуемых форм-компонентов

Аналогичные проблемы могут быть в:
- `CreateGroupModal.tsx`
- `LoginPage.tsx`
- `RegisterPage.tsx`
- `ForgotPasswordPage.tsx`
- `ProfilePage.tsx`
- `SettingsPage.tsx`

## Цель задачи
Внедрить `react-hook-form` или создать оптимизированные обработчики форм для улучшения производительности и DX.
