# Задача 08: Управление записями и медиа-альбомы

## Описание
Расширение возможностей создания и редактирования записей для соответствия различным типам контента (книги, фильмы, сериалы) и добавление функционала медиа-альбомов.

## Цель задачи
Сделать формы ввода гибкими и контекстно-зависимыми, а также дать пользователю возможность прикреплять медиа-контент к тайтлам.

## Чек-лист выполнения

### Этап 1: Создание и редактирование записей
- [x] **Обложка**: Реализовать переключатель типа загрузки обложки (Загрузка файла / Ссылка URL).
- [x] **Динамические поля**: Реализовать отображение полей в зависимости от категории:
  - [x] *Книга/Манга*: "Дата начала чтения", "Дата окончания чтения".
  - [x] *Фильм*: "Дата просмотра".
  - [x] *Сериал*: "Дата начала просмотра", "Дата окончания просмотра".
- [x] **Критерии**: Добавить выбор дополнительных критериев из списка (Tags/Attributes).
- [x] **Инструменты**: Реализовать кнопки "Редактировать", "Сохранить", "Удалить" в интерфейсе просмотра.

### Этап 2: Раздел "Альбом"
- [x] **Вкладка**: Добавить вкладку или секцию "Альбом" в детальном просмотре записи.
- [x] **Загрузка**: Реализовать загрузку фотографий и видео файлов.
- [x] **Галерея**: Реализовать просмотр медиа-альбома (Grid view).
- [x] **Управление**: Добавить возможность удаления и просмотра медиафайлов (Lightbox/Modal).

## Критерии приёмки
- ✅ Форма создания адаптируется под выбранную категорию.
- ✅ Можно загрузить обложку как файлом, так и по ссылке.
- ✅ Дата просмотра/чтения сохраняется корректно.
- ✅ Медиафайлы успешно загружаются и отображаются в альбоме тайтла.

## Технические требования
- Оптимизация загрузки больших изображений.
- Валидация типов файлов (Image/Video).
