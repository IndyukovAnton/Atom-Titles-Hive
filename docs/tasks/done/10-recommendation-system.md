# Задача 10: Система рекомендаций

## Описание
Создание интеллектуальной системы рекомендаций на основе предпочтений пользователя и интеграции с нейросетями.

## Цель задачи
Повысить вовлеченность пользователя, предлагая релевантный контент.

## Чек-лист выполнения

### Этап 1: Алгоритмические рекомендации
- [x] **Раздел UI**: Создать страницу или блок "Рекомендации".
- [x] **История**: Реализовать подбор на основе истории просмотров.
- [x] **Жанры**: Реализовать подбор по предпочитаемым жанрам.
- [x] **Рейтинг**: Предлагать тайтлы с высоким рейтингом (Top Rated).
- [x] **Объем**: Учитывать популярность жанра в библиотеке.
- [x] **Новинки**: Предлагать непросмотренные/отсутствующие в базе тайтлы.

### Этап 2: Интеграция с нейросетями (AI)
- [x] **Провайдеры**: Реализовать интерфейс выбора нейросети (Predefined options + Custom).
- [x] **Настройка**: Добавить поле для ввода API Key и лимитов использования.
- [x] **Генерация**: Реализовать запрос к AI для генерации подборки.
- [x] **Импорт**: Реализовать кнопку "Быстрое добавление" тайтла из рекомендаций в базу.

## Критерии приёмки
- ✅ Раздел рекомендаций отображает списки по разным критериям.
- ✅ Можно подключить свой API ключ для AI.
- ✅ AI генерирует адекватные советы.
- ✅ Можно добавить тайтл в библиотеку в один клик.

## Технические требования
- Интеграция с внешними API через Proxy (backend) для безопасности ключей.
- Кэширование результатов рекомендаций.
