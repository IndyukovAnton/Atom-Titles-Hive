# Задача 11: Персонализация и настройки

## Описание
Внедрение глубокой персонализации интерфейса.

## Цель задачи
Позволить пользователю "настроить приложение под себя", включая визуальный стиль и параметры поведения.

## Чек-лист выполнения

### Этап 1: Сбор данных
- [x] **Дата рождения**: Реализовать запрос даты рождения при первом входе (влияет на рекомендации).

### Этап 2: Визуальная кастомизация
- [x] **Фон**: Реализовать выбор стиля заднего фона рабочей области (10 вариантов):
  - [x] Liquid Ether
  - [x] Light Pillar
  - [x] Aurora
  - [x] 7 дополнительных вариантов (Sunset, Forest, Midnight, Cosmos, Ocean, Passion, Default).
- [x] **Шрифты**:
  - [x] Настройка размера шрифта.
  - [x] Выбор семейства шрифта (из Google Fonts).
- [x] **Язык**: Переключатель языка интерфейса (i18n) - база готова, можно расширить.

### Этап 3: Настройки AI
- [x] **UI Настроек**: Секция для ввода параметров нейросетей (Provider, Key, Limits) для использования в модуле рекомендаций.

## Критерии приёмки
- ✅ Выбранный фон применяется ко всему приложению.
- ✅ Размер шрифта корректно масштабирует интерфейс.
- ✅ Настройки сохраняются в профиле пользователя.

## Технические требования
- Использовать Context API или Global Store для прокидывания настроек темы.

## Реализовано
- Создан ProfileModule на бэкенде с API эндпоинтами
- Расширена User entity с полями preferences, birthDate, hasCompletedOnboarding
- Создана миграция для БД
- Создан PersonalizationContext для управления настройками
- Компонент BackgroundSelector с 10 вариантами фонов
- Компонент FontSettings для выбора шрифта и размера из Google Fonts
- Компонент AISettings для настройки AI провайдера
- Полностью обновлена страница SettingsPage
- Все настройки сохраняются в профиле через API
