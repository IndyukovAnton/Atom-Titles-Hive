# Задача 13: Обучение и визуальные эффекты

## Описание
Полировка UI и внедрение интерактивного обучения для новых пользователей.

## Цель задачи
Создать "Wow-эффект" при первом знакомстве и обучить функциям приложения.

## Чек-лист выполнения

### Этап 1: Обучение (Onboarding)
- [x] **Stepper**: Реализовать пошаговое руководство с использованием анимаций и прогресс-бара.
- [x] **Сценарий**: Описать основные функции (Создание записи, Папки, Поиск).
- [x] **Контроль**: Кнопки "Пропустить", "Повторить", "Далее".

### Этап 2: Визуальные эффекты (Wow-factors)
- [x] **Загрузка**: Анимированная заставка при старте (Video Text) с фоновой загрузкой данных.
- [x] **Страницы**: Улучшить дизайн Регистрации и Главной (Градиенты, Тени, Анимации появления).
- [x] **Пасхалка**:
  - [x] Использовать падающие буквы с анимацией.
  - [x] Триггер: Тройное нажатие на заголовок на главной странице.

## Критерии приёмки
- ✅ Новый пользователь видит обучение.
- ✅ Заставка плавная, без рывков.
- ✅ Пасхалка работает.
- ✅ Интерфейс выглядит современно и "дорого".

## Технические требования
- Оптимизация тяжелых анимаций (CSS GPU acceleration).

## Реализовано
- Компонент OnboardingFlow с пошаговым руководством, анимациями и прогресс-баром
- Компонент LoadingSplash с анимированной заставкой и частицами
- Улучшен дизайн RegisterPage с градиентами, анимациями и современным стилем
- Пасхалка FallingText активируется тройным кликом на заголовок HomePage
- Интеграция в App.tsx с автоматическим показом онбординга для новых пользователей
- Добавлены CSS анимации с GPU acceleration для плавности
- Уведомления через Sonner для лучшего UX
