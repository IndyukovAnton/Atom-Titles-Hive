# Задача 11: Персонализация и настройки

## Описание
Внедрение глубокой персонализации интерфейса.

## Цель задачи
Позволить пользователю "настроить приложение под себя", включая визуальный стиль и параметры поведения.

## Чек-лист выполнения

### Этап 1: Сбор данных
- [x] **Дата рождения**: Реализовать запрос даты рождения при первом входе (влияет на рекомендации).

### Этап 2: Визуальная кастомизация
- [x] **Фон**: Реализовать выбор стиля заднего фона рабочей области (10 вариантов):
  - [x] Liquid Ether
  - [x] Light Pillar
  - [x] Aurora
  - [x] 7 дополнительных вариантов.
- [ ] **Шрифты**:
  - [ ] Настройка размера шрифта.
  - [ ] Выбор семейства шрифта (из Google Fonts).
- [ ] **Язык**: Переключатель языка интерфейса (i18n).

### Этап 3: Настройки AI
- [ ] **UI Настроек**: Секция для ввода параметров нейросетей (Provider, Key, Limits) для использования в модуле рекомендаций.

## Критерии приёмки
- ✅ Выбранный фон применяется ко всему приложению.
- ✅ Размер шрифта корректно масштабирует интерфейс.
- [x] ✅ Настройки сохраняются в профиле пользователя.

## Технические требования
- Использовать Context API или Global Store для прокидывания настроек темы.
