# Задача 13: Обучение и визуальные эффекты

## Описание
Полировка UI и внедрение интерактивного обучения для новых пользователей.

## Цель задачи
Создать "Wow-эффект" при первом знакомстве и обучить функциям приложения.

## Чек-лист выполнения

### Этап 1: Обучение (Onboarding)
- [ ] **Stepper**: Реализовать пошаговое руководство с использованием `Stepper` из `@React Bits`.
- [ ] **Сценарий**: Описать основные функции (Создание записи, Папки, Поиск).
- [ ] **Контроль**: Кнопки "Пропустить", "Повторить", "Далее".

### Этап 2: Визуальные эффекты (Wow-factors)
- [ ] **Загрузка**: Анимированная заставка при старте (Video Text) с фоновой загрузкой данных.
- [ ] **Страницы**: Улучшить дизайн Регистрации и Главной (Градиенты, Тени, Анимации появления).
- [ ] **Пасхалка**:
  - [ ] Использовать `Falling Text` из `@React Bits`.
  - [ ] Триггер: Тройное нажатие на заголовок в Политике конфиденциальности.

## Критерии приёмки
- ✅ Новый пользователь видит обучение.
- ✅ Заставка плавная, без рывков.
- ✅ Пасхалка работает.
- ✅ Интерфейс выглядит современно и "дорого".

## Технические требования
- Оптимизация тяжелых анимаций (CSS GPU acceleration).
