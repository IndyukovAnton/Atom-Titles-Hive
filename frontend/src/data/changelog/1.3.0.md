Большое обновление, ради которого затеян релиз 0.3.0. Главное — настоящие персональные рекомендации через Claude (как через локально установленный CLI, так и через API key), новые разделы для коллекций и работа в фоне.

## AI рекомендации

- Раздел **Рекомендации → AI Assistant** теперь выдаёт настоящие персональные подборки от Claude, опираясь на всю вашу библиотеку.
- Источник выбирается в **Настройках → AI**: «Claude (Cloud API)» — через ваш Anthropic API ключ; либо «Claude (Local CLI)» — через локально установленный `claude` (без передачи данных в облако).
- Карточки **появляются по мере генерации**: видно прогресс размышлений модели, сколько уже готово, и текущее действие («Анализирую библиотеку», «Ищу в вебе свежие релизы»).
- Поиск **актуальных новинок** через встроенный WebSearch — релизы за последние месяцы попадают в подборку.
- На каждой карточке постер (через ваш TMDB ключ если задан, иначе через встроенный поиск изображений), badge «NEW» для свежих релизов, оценка от модели и обоснование почему именно вам.
- Жанры выбираются **тегами**: 18 пресетов (Боевик, Sci-Fi, Slice of Life и т.п.). Можно комбинировать с настроением и типом контента.
- **«Новое для меня»** — переключатель в форме. Когда включён, Claude сознательно предлагает жанры, которых **нет** в вашей библиотеке — для расширения кругозора.
- Результаты **сохраняются**: обновили страницу, закрыли приложение — карточки остались на месте до 24 часов или до следующего запроса.
- Повторные запросы не предлагают то же самое: предыдущие карточки автоматически исключаются из новой подборки.
- Если что-то пошло не так — в блоке ошибки кнопка «Подробнее» показывает stderr / exit code / argv для диагностики.
- Страница рекомендаций теперь во всю ширину экрана, форма слева остаётся на виду при прокрутке.

## «Подумаю» и «Избранное»

- В сайдбаре два новых раздела: **📌 Подумаю** и **⭐ Избранное**.
- На любой AI-рекомендации — кнопка 📌 кладёт её в «Подумаю», ⭐ — в «Избранное». Не нужно сразу решать, добавлять в библиотеку или нет.
- В библиотеке на каждой карточке записи появилась ⭐ — отметить как избранное.
- **Избранное** объединяет в одном месте и записи из библиотеки, и сохранённые рекомендации. Есть табы: «Все» / «Библиотека» / «Рекомендации».
- Из «Подумаю» можно одним кликом отправить в «Избранное» или в библиотеку, или удалить.

## Работа в фоне

- Закрытие окна крестиком **больше не выходит из приложения** — оно сворачивается в системный трей. Backend продолжает работать, рекомендации продолжают генерироваться.
- Иконка в трее: левый клик — показать окно. Правый клик — меню «Открыть Seen» / «Выйти».
- Когда рекомендации готовы и окно скрыто — приходит **системное уведомление** «Рекомендации готовы: N карточек за Xс».

## Защита от типичных AI-промахов

- Системный промпт жёстко требует от модели опираться на жанры и настроения, которые **уже есть** в вашей библиотеке, и не придумывать рекомендации в чужих орбитах. Это убирает классическую ситуацию «попросил sad mood, получил случайный depressing horror, который мне никогда не нравился». Расширение горизонта — только через явный переключатель «Новое для меня».
- Кеш контекста: повторный запрос внутри 10 минут не делает заново разбор библиотеки — быстрее.

## Настройки → AI

- Селектор источника карточками (Cloud API / Local CLI) с состоянием каждого.
- Для CLI: живой статус «установлен / не авторизован / готов» с кнопкой «Re-check» и инструкциями как исправить.
- Дополнительные провайдеры (OpenAI / Gemini / Cohere / Local) собраны в сворачиваемый блок — заделом на будущее.

## Прочее

- Кнопка «Назад» в Настройках теперь действительно возвращает на предыдущую страницу, а не на главную.
- Ширина и расстояния в форме рекомендаций — выровнены: настроения, жанры, типы и количество больше не сжимаются.
- Текст обоснования рекомендации виден целиком, без обрезки.
